# MLB Player Salary Prediction System

## Overview
This project develops machine learning and statistical modeling workflows to predict Major League Baseball player salaries using historical player performance metrics, offensive statistics, and career production data. The analysis focuses on preprocessing pipelines, feature engineering, regression modeling, and analytical evaluation techniques to identify the variables most strongly associated with player market valuation.

## Objectives
- Predict MLB player salaries using historical player performance datasets
- Develop preprocessing and feature engineering workflows for sports analytics applications
- Analyze relationships between player statistics and salary valuation
- Compare predictive modeling approaches across structured baseball datasets
- Build scalable analytical workflows for player valuation and salary forecasting

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
- Regression Modeling
- Feature Engineering
- Data Preprocessing
- Statistical Analysis

## Key Features
- Built preprocessing and validation pipelines for historical MLB player performance and salary datasets
- Engineered structured analytical features using offensive production, career statistics, and player performance metrics
- Applied regression modeling workflows to estimate player salary valuation and identify high-impact performance indicators
- Conducted exploratory statistical analysis to determine which player metrics were most strongly associated with salary outcomes
- Developed repeatable training, testing, and evaluation workflows to improve predictive consistency and model reliability

## Results
The project demonstrated how structured preprocessing, feature engineering, and regression modeling workflows can improve salary prediction performance while identifying the player performance variables most strongly associated with MLB market valuation and compensation trends.
